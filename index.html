Tetris AImore_vert
close 2016

Developer
In this project we built a Tetris AI using an utility function based on a linear weighted sum of heuristics. The parameters for the heuristics were obtained by running a genetic algorithm and the AI improved its skill by replacing the current best values with new ones by running multiple evolutions on different threads. The AI managed to clear an average of 2 million lines per game.
